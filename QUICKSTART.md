# 心动 - 快速开始指南

## 🚀 立即体验

应用已经在您的浏览器中运行！

**访问地址：** http://localhost:5174/

## 📱 功能概览

### 1. 主界面
- **顶部栏**：显示连接状态、音效开关、连接和设置按钮
- **心率显示**：中央大字体显示当前心率（BPM）
- **心电图**：下方实时绘制 ECG 波形
- **3D背景**：动态粒子效果，会随心率变化

### 2. 连接设备
1. 点击"连接设备"按钮
2. 在弹出的蓝牙设备列表中选择您的心率设备
3. 等待连接成功
4. 开始实时监测心率

### 3. 设置选项
点击"设置"按钮可以：
- 切换深色/浅色主题
- 自定义主题颜色
- 查看使用说明

## 💡 使用提示

### 没有蓝牙设备？
没关系！您可以：
1. 查看界面布局和视觉效果
2. 体验主题切换功能
3. 观察心电图的动画效果（未连接时显示基线）

### 有蓝牙心率设备？
请确保：
1. 设备已开启"心率广播"或"心率共享"功能
2. 设备未连接到其他应用
3. 设备电量充足

### 浏览器兼容性
- ✅ **推荐**：Chrome、Edge（最新版）
- ❌ **不支持**：Firefox、Safari

## 🎨 自定义

### 更改主题色
1. 点击"设置"
2. 在"外观设置"中选择喜欢的颜色
3. 或使用预设颜色快速切换

### 切换明暗主题
1. 点击"设置"
2. 选择"浅色模式"或"深色模式"

### 核心功能实现

#### 蓝牙连接
使用 Web Bluetooth API 连接标准心率设备：
```typescript
navigator.bluetooth.requestDevice({
  filters: [{ services: ['heart_rate'] }]
})
```

#### 心率数据解析
从蓝牙特征值中解析心率：
```typescript
const flags = value.getUint8(0)
const heartRate = flags & 0x1 
  ? value.getUint16(1, true)  // 16位
  : value.getUint8(1)          // 8位
```

#### 心电图绘制
基于心率生成模拟的ECG波形，包含P波、QRS复合波、T波。

#### 3D粒子背景
使用Three.js创建2000个粒子，根据心率调整旋转速度和缩放。
